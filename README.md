# Meridian Studio - Custom WordPress Theme

A minimalist and fully responsive custom WordPress theme built from scratch for Meridian Studio.

## Features
- **Custom WordPress Loop:** Dynamically loads journal entries.
- **Responsive Layout:** Optimized for desktop, tablet, and mobile screens.
- **Custom Page Templates:** Tailored layouts for Home, About, Services, Journal, and Contact pages.

## Technologies Used
- WordPress (PHP)
- HTML5 & CSS3
